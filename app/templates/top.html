Remind Till Done

A Slack App to add reminders to members of channels.

How to use

/remind-till-done command add a Done button to the channel,
and add reminders to members of the channel.
Press the Done button then remove the reminder.

reminding time

To set the reminder time, send command with a string like “Today is January 1, 2047 at 8:21:00AM”.
If it is invalid or nothing, set tomorrow at the time.
